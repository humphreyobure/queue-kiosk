# Queue Management System - Kiosk Interface

Touch-friendly self-service kiosk for patient check-in.

## Tech Stack
- Vue.js 3
- Vite
- Tailwind CSS

## Features
- Touch-optimized UI
- Service selection
- Ticket generation
- Print preview
- Multi-language support ready

## Setup
```bash
npm install
npm run dev
```
